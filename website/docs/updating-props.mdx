React components use [props](https://facebook.github.io/react-native/docs/props) to recieve data when they are created.
When displaying [component]() layout, you can pass initial props to the component with the `passProps` property.

To update a component's props, use the [Navigation.updateProps()]() command. Updating props triggers
the component lifecycle methods related to [updating](https://reactjs.org/docs/react-component.html#updating).
