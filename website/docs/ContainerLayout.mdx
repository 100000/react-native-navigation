Extends [Layout](layout)

## `id`

| Type   | Required | Description                                                                                                                                                    |
| ------ | -------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| string | No       | Unique id used when interacting with the view with the Navigation api, usually `Navigation.mergeOptions` which accepts the componentId as it's first argument. |

## `children`

| Type               | Required | Description                   |
| ------------------ | -------- | ----------------------------- |
| [Layout[]](layout) | YES      | Children layouts of any kind. |

## `options`

| Type                    | Required | Description                                              |
| ----------------------- | -------- | -------------------------------------------------------- |
| [Options](options-root) | No       | dynamic options which will apply to all children layouts |
